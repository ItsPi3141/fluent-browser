Simple web browser made with ElectronJS, featuring a mica theme

### How to run:

Clone repo
```
git clone https://github.com/ThePiGuy3141/electron-mica-browser.git
cd electron-mica-browser
```

Install node modules
```
npm i
```

Start the app
```
npm start
```
